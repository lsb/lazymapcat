# lazymapcat

A Clojure library designed to provide a lazier mapcat than default.

Kudos to http://clojurian.blogspot.com/2012/11/beware-of-mapcat.html

## Usage

Leiningen: [lazymapcat "0.1.0"]

(lazymapcat/mapcat f [:this :that :the-other])
